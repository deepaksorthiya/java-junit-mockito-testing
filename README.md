# Getting Started

## Requirements:

```
JDK: 25
Maven: 3.9+
JUnit: 6+
```

### Clone this repository:

```bash
git clone https://github.com/deepaksorthiya/java-junit-mockito-testing.git
```

```bash
cd java-junit-mockito-testing
```

### Run Test(Maven Should Be Installed):

```bash
mvn clean test
```
